## secret create

```yaml
kubens playground-1

# Create a new secret named my-secret with keys for each file in folder bar
kubectl create secret generic my-secret-file --from-file=something-bar

#Create a new secret named my-secret with key1=supersecret 
kubectl create secret generic my-secret-literal --from-literal=key1=supersecret 

#Create a new secret named my-secret from an env file
kubectl create secret generic my-secret-env --from-env-file=something-bars.env

```
